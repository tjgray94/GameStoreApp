About GameStore

Welcome to GameStore, your premier destination for discovering and managing your video game collection. Developed by Tevin Gray, this application showcases modern web development practices while providing a seamless gaming inventory experience.

GameStore allows you to browse, add, edit, and organize your favorite games with ease. Whether you're a casual gamer or a dedicated collector, our platform helps you keep track of your gaming library with detailed information about each title.

Key Features

  • Browse and search through our extensive game catalog
  • Add new games to the collection with detailed information
  • Edit game details including ratings, release dates, and prices
  • Rate games and track your favorites
  • User account management for personalized experiences

Technologies Used

GameStore is built using modern web development technologies, demonstrating full-stack development principles:

Angular TypeScript Bootstrap Node.js RESTful API Responsive Design
